# Debug实验室物联网
## 说明
- 设备基于arduino UNO和ESP8266wifi开发，包含个模块的测试程序继承在里面；
- 此版本在vs2017+arduino IDE for VisualStudio环境下编译通过；
- 所用到的硬件：

	| 设备名称        |  数量  |
    | -----------------| :-----: |
    |arduino UNO   |1       |
    | ESP8266        | 1      |
    | 5V继电器       | 1      |
    |面包板线|若干|
   
- 使用时可能需要的库安装方法自行Google：
	- ArduinoJson
	- ESP8266WiFi.h
- EEPROM中预置信息暂无软件写入，可以通过testEEPROM调试写入；
- ESP8266的连接需要手动设置；
- 该版本只能算是测试用半成品，离开开发环境基本不能使用，仅作为参考使用；
